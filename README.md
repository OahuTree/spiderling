# Spiderling - Web Scraper (网页内容抓取器)

[English](README.md) | [简体中文](docs/README_ZH.md)

Spiderling is a simple, efficient, and automated web scraping tool built with Python and PyQt5. It provides a user-friendly graphical interface to define, manage, and execute complex scraping workflows without deep coding knowledge.
