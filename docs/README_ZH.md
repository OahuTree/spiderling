# Spiderling - 网页内容抓取器

[English](../README.md) | [简体中文](README_ZH.md)

Spiderling 是一款基于 Python 和 PyQt5 构建的简单、高效、自动化的网页内容采集工具。它提供了一个友好的图形化界面，让用户无需深入的代码知识即可定义、管理和执行复杂的爬虫工作流。

## 🚀 功能特性

- **直观的工作流管理**：将抓取任务定义为一系列有序的步骤（动作）。
- **丰富的动作集合**：
  - **定位导航**：打开 URL，在网页间跳转。
  - **交互操作**：点击元素、输入文本、执行自定义 JavaScript。
  - **数据提取**：通过 CSS 选择器解析网页、解析本地 HTML 文件或从特定 URL 提取信息。
  - **逻辑控制**：支持循环（通过缓存迭代）、跳转至指定步骤、重置流程等。
  - **延时等待**：内置等待机制，轻松应对动态加载的内容。
- **可定制的数据转换**：
  - **百分数转换**：将百分数字符串（如 "15%"）转换为浮点数（0.15）。
  - **千分位清理**：取消数字中的千分位（逗号）分隔符。
  - **进阶**：可以DIY通过配置文件完成数据转换。
- **多数据库支持**：可将抓取的数据无缝存储至 **SQLite、MySQL、PostgreSQL、SQL Server 或 Oracle**。
- **浏览器管理**：集成清空 Chrome 缓存和强制清理浏览器进程的功能。
- **多语言支持**：内置 **简体中文、繁体中文、英语、日语**。
