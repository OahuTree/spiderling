# Spiderling - ウェブスクレイパー (Web Scraper)

[简体中文](README_zh.md) | [繁體中文](README_tw.md) | [日本語](README_ja.md) | [日本語 (和風)](README_ja_trad.md) | [English](../README.md)  


Spiderlingは、PythonとPyQt5で構築された、シンプルで効率的、そして自動化されたウェブコンテンツ収集ツールです。深いプログラミングの知識がなくても、複雑なスクレイピングワークフローを定義、管理、実行できる使いやすいグラフィカルインターフェースを提供します。

![dashboard.png](images/dashboard.png)

## 🚀 主な機能

- **直感的なワークフロー管理**: スクレイピングタスクを一連の順序付けられたステップ（アクション）として定義。
- **豊富なアクションセット**:
  - **ナビゲーション**: URLを開き、ウェブページ間を移動。
  - **インタラクション**: 要素のクリック、テキストの入力、カスタムJavaScriptの実行。
  - **データ抽出**: CSSセレクターを使用したページ解析、ローカルHTMLの解析、または特定のURLからの情報抽出。
  - **ロジック制御**: ループ（キャッシュ経由の反復）、指定ステップへのジャンプ、フローのリセットなどをサポート。
  - **遅延と待機**: 動的コンテンツの読み込みに対応する待機メカニズムを内蔵。
- **カスタマイズ可能なデータ変換**:
  - **パーセント変換**: パーセント文字列（例: "15%"）を浮動小数点数（0.15）に変換。
  - **桁区切り解除**: 数値データから千の位の区切り文字（カンマ）を削除。
  - **高度な設定**: DIY構成ファイルを介してデータ変換を完了可能。
- **マルチデータベース対応**: 収集したデータを **SQLite, MySQL, PostgreSQL, SQL Server, Oracle** にシームレスに保存。
- **ブラウザ管理**: Chromeキャッシュのクリアとブラウザプロセスの強制終了機能を統合。
- **多言語対応**: **日本語、簡体字中国語、繁体字中国語、英語** を内蔵。

## 📦 インストール方法

### 環境要件
- Python 3.10+
- Google Chrome ブラウザがインストールされていること

### インストール手順
1. **リポジトリのクローン**:
   ```bash
   git clone https://github.com/OahuTree/spiderling.git
   ```

2. **依存関係のインストール**:
   ```bash
   pip install -r requirements.txt
   ```

3. **プログラムの起動**:
   ```bash
   python spiderling.py
   ```

## 📖 使用ガイド
1. **ブラウザ設定**: プログラムが自動認識しない場合は、設定でChromeの実行ファイルパスを指定してください。
![dashboard.png](images/browser_config.png)
2. **データベース設定**: 「データベース設定」タブでターゲットとなるデータベース接続を設定します。
3. **ステップの定義**:
   - 新しいワークシートを追加。
   - 「ステップ追加」ダイアログを使用してスクレイピングロジックを定義。
   - アクションタイプ（「ウェブページ移動」、「要素クリック」、「ウェブ解析」など）を選択し、対応するパラメータ（URL、CSSセレクターなど）を入力。
   - [spiderling_example.xlsx](spiderling_example.xlsx) を参考にしてください。
4. **データ変換**: 保存前に「データ準備 (Stage Data)」アクションを使用してデータ変換ルールを適用します。
5. **タスクの実行**: 「スクレイピング開始」をクリックして自動化プロセスを起動します。下部の「ログセンター」で進行状況を監視できます。
6. **ディレクトリの読み書き権限**: プログラムは現在のログインユーザーのディレクトリに 'oahutree_spiderling' フォルダを作成します。プログラムにこのディレクトリの読み書き権限があることを確認してください。


## ⚙️ 設定ファイルについて
