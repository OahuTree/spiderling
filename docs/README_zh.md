# Spiderling - 网页内容抓取器

[English](../README.md) | [简体中文](README_zh.md)

Spiderling 是一款基于 Python 和 PyQt5 构建的简单、高效、自动化的网页内容采集工具。它提供了一个友好的图形化界面，让用户无需深入的代码知识即可定义、管理和执行复杂的爬虫工作流。

## 🚀 功能特性

- **直观的工作流管理**：将抓取任务定义为一系列有序的步骤（动作）。
- **丰富的动作集合**：
  - **定位导航**：打开 URL，在网页间跳转。
  - **交互操作**：点击元素、输入文本、执行自定义 JavaScript。
  - **数据提取**：通过 CSS 选择器解析网页、解析本地 HTML 文件或从特定 URL 提取信息。
  - **逻辑控制**：支持循环（通过缓存迭代）、跳转至指定步骤、重置流程等。
  - **延时等待**：内置等待机制，轻松应对动态加载的内容。
- **可定制的数据转换**：
  - **百分数转换**：将百分数字符串（如 "15%"）转换为浮点数（0.15）。
  - **千分位清理**：取消数字中的千分位（逗号）分隔符。
  - **进阶**：可以DIY通过配置文件完成数据转换。
- **多数据库支持**：可将抓取的数据无缝存储至 **SQLite、MySQL、PostgreSQL、SQL Server 或 Oracle**。
- **浏览器管理**：集成清空 Chrome 缓存和强制清理浏览器进程的功能。
- **多语言支持**：内置 **简体中文、繁体中文、英语、日语**。

## 📦 安装说明

### 环境要求
- Python 3.10+
- 已安装 Google Chrome 浏览器

### 安装步骤
1. **克隆仓库**：
   ```bash
   git clone https://github.com/OahuTree/spiderling.git
  

2. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```

3. **启动程序**：
   ```bash
   python spiderling.py
   ```

## 📖 使用指南
1. **浏览器配置**：如果程序未自动识别，请在设置中指定 Chrome 浏览器的可执行文件路径。
2. **数据库设置**：在“数据库配置”选项卡中设置目标数据库连接。
3. **定义步骤**：
   - 新增工作表。
   - 使用“添加步骤”对话框定义抓取逻辑。
   - 选择动作类型（如“定位网页”、“点击控件”、“网页解析”），并填写相应参数（URL、CSS 选择器等）。
   - 可以参考[spiderling_example.xlsx](spiderling_example.xlsx)
4. **数据转换**：在保存前使用“准备数据 (Stage Data)”动作应用数据转换规则。
5. **执行任务**：点击“开始抓取”启动自动化流程。您可以在其底部的“日志中心”监控运行进度。
